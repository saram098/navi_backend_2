Note: This is a development server. Use the following command to run the FastAPI application:

python run_fastapi.py
API Documentation Interactive Swagger UI

About the API

This backend system provides a comprehensive solution for clinic appointment management and patient interaction through WhatsApp chatbot integration.

Key Features

Authentication User Management

Secure user registration, login, email verification, and password management.

Appointments Scheduling System

Book, view, update, and cancel appointments with physicians.

Chatbot WhatsApp Integration

AI-powered chatbot for appointment booking, checking availability, and answering questions via WhatsApp.

Payments Secure Transaction Processing

Integrated payment processing using Stripe for appointment fees.

Insurance Verification System

Check insurance coverage and eligibility for medical services.

Technology Stack

  • Backend Framework: FastAPI
  • Database: MongoDB
  • External Services: Twilio (WhatsApp), OpenAI (Chatbot Intelligence), Stripe (Payments)
  • Authentication: JWT (JSON Web Tokens)

Running the API

To run the FastAPI application with full functionality, use either:

python run_fastapi.py
# OR
python -m uvicorn main:create_fastapi_app --host 0.0.0.0 --port 5000

The FastAPI interactive documentation will be available at /docs when running the FastAPI application.
